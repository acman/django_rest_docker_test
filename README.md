# Django REST API + Docker test project

Login/Pass: admin@admin.com/adminadmin1

Run project:
```
$ git clone git@github.com:acman/django_rest_docker_test.git
$ cd django_rest_docker_test
$ make
```

Build docker image:
```
$ make build
```

Run server:
```
$ make run
```

Stop server:
```
$ make stop
```
